
# VUNex AI 문서

이 문서는 **이노뎁(innodep)의 엣지형 영상 분석 장치 VUNex AI**의 매뉴얼입니다.

## 제품 개요
VUNex AI는 **실시간 메타데이터 분석**, **적응형 AI 모드**, **고급 검색 기능**을 제공하는 엣지형 영상 분석 장치입니다. 

HTML5 기반 UI를 통해 플러그인 없이 쉽게 설정할 수 있으며, 엣지에서 이벤트를 분석하여 네트워크 트래픽과 서버 부하를 절감할 수 있습니다.

## 주요 기능
- **실시간 메타데이터 분석**: 다중 채널 데이터를 실시간으로 분석하고 시각화합니다.
- **적응형 AI 모드**: 산업별 특화된 AI 모드를 제공하며, 사용자 맞춤 설정이 가능합니다.
- **고급 검색 기능**: Re-ID 기반 경로 추적, VLM 기반 지능형 검색 등 상세한 검색 기능을 지원합니다.
- **엣지 디바이스 효율성**: 이벤트를 엣지에서 분석하여 네트워크 트래픽과 서버 부하를 줄일 수 있습니다.
- **HTML5 기반 UI**: 플러그인 없이 웹 브라우저에서 간편하게 설정할 수 있습니다.
